	0102_What Is Materialize & Why Use It?
	- What is Maaterialize?
		- Modern framework based on Material Design
		- Similar to Bootstrap, Foundation, etc
		- Interactive Components
		- Extremely Customizable
	- Material Design
		- Design language/scheme developed by Google in 2014
		- Makes liberal use of grid layouts, responsive animations, shadows, etc
		- Based on "paper & ink"
		- Most Google based UIs use the material design concepts
	- Why learn Materialize?
		- Build beautiful websites & app interfaces with little design skill
		- MUCH more than just a grid
		- 100% responsive by design
		- Great looking & easy to implement CSS components
		- Many interactive JavaScript widgets and functionality
	- Version info
		- This course will use v.0.100.2(materializecss.com)
		- 1.0 is in alpha and when it is released the course will be updated
		- There will not be many changes
